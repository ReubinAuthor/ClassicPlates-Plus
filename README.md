# **Reubin's Nameplates**
Reubin's nameplates is a simple Classic-themed addon that provides additional information about everyone around you.

<br />

Features:
- Shows Health
- Shows Portraits and class icons
- Shows power bar
- Shows Threat status icon and and percentage
- Shows Buffs and Debuffs applied by you
- Shows NPC's classification (Elite, Rare, Rare elite, World Boss)
- Shows Health bar colors
- Lets you blacklist auras you don't want to keep track off
- Highlights whoever is targeting you
- Lets you change nameplates size
- Tank mode - Shows you which NPCs are running loose and which are tanked by you or another tank.
